🌍 战斗场景 (5个)

  2. 熔岩炼狱 - 炽热的火山地带，充满危险

⚔️ 战斗规则

  • 靠近对手后使用攻击键造成伤害
  • 防御可以减少 50% 受到的伤害（部分角色有特殊防御效果）
  • 每个角色有不同的属性和特色能力
  • 先将对方血量降为 0 的一方获胜
  • 点击"再来一局"可以快速重赛

🚀 快速开始

方法一：直接打开

直接在浏览器中打开 index.html 文件即可游玩。

方法二：使用本地服务器

# Python 3
python3 -m http.server 8080
# 然后在浏览器中访问 http://localhost:8080/index.html

方法三：使用 Node.js

# 安装依赖
npm install

# 启动开发服务器
npm run dev

🎯 技术特点

  • 纯 HTML5 Canvas + JavaScript 实现，无需任何框架
  • MVVM 架构模式，代码组织清晰，易于维护
  • 像素风格渲染，支持角色朝向翻转
  • 完整的游戏状态管理
  • 特殊能力系统实现
  • 响应式界面设计
  • 流畅的 60fps 游戏循环
  • TypeScript 类型支持（可选）

💡 扩展建议

提示：由于采用了 MVVM 架构，后续可以轻松添加以下功能：
  • AI对战 - 创建 AIViewModel 处理电脑控制
  • 联机模式 - 通过 Service 层实现网络通信
  • 音效系统 - 添加 AudioService 服务
  • 成就系统 - 在 ViewModel 层添加统计追踪
  • 动画系统 - 扩展 RenderService 支持更复杂的动画

⚔️ Pixel Mech Battle - Ultimate Edition

A retro pixel-style two-player battle game with 10 unique characters and 5 battle scenes!

🎮 10 Characters

Each character has unique stats and special abilities

🌍 5 Scenarios

From martial arts dojo to space battlefield

⚡ Special Skills

Fire, ice, poison and more combat experiences

🏆 MVVM Architecture

Modular design, easy to extend and maintain

🎮 Game Features

  • 10 Unique Mech Characters - Each character has different stats and special abilities
  • 5 Battle Scenes - From martial arts dojo to space battlefield, each with unique characteristics
  • Special Ability System - Each character has unique combat skills
  • Complete Game Flow - Main Menu → Character Selection → Scene Selection → Battle
  • MVVM Architecture - Modern engineering design, easy to maintain and extend

🕹️ Controls

Player 1 (Green)

  • W - Move Up
  • S - Move Down
  • A - Move Left
  • D - Move Right
  • F - Attack
  • G - Defend

Player 2 (Purple)

  • ↑ - Move Up
  • ↓ - Move Down
  • ← - Move Left
  • → - Move Right
  • . (Period) - Attack
  • / (Slash) - Defend

⚔️ Battle Rules

  • Get close to your opponent and use the attack key to deal damage
  • Defending reduces damage taken by 50% (some characters have special defense effects)
  • Each character has different stats and special abilities
  • The player who reduces the opponent's health to 0 first wins
  • Click "Rematch" for a quick restart
